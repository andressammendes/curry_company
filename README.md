# Curry Company — Data Analysis & Delivery Performance Dashboard

## Português

### 📌 Sobre o projeto

Este projeto consiste em uma análise completa de dados de uma operação de delivery, com foco em identificar os principais fatores que impactam o tempo de entrega e a eficiência operacional.

A solução foi desenvolvida de ponta a ponta, incluindo:

* Limpeza e tratamento de dados
* Análise exploratória
* Construção de métricas de negócio
* Desenvolvimento de dashboard interativo
* Deploy em ambiente web

🔗 Acesse o dashboard:
https://amendes-curry-company.streamlit.app/

---

### 🎯 Objetivo

Responder à seguinte pergunta de negócio:

> Quais fatores mais impactam o tempo de entrega e onde estão os principais gargalos operacionais?

---

### 📊 Principais análises

* Impacto da densidade de tráfego no tempo de entrega
* Comparação entre diferentes cidades
* Influência das condições climáticas
* Análise de performance dos entregadores
* Impacto de eventos (festivais) na operação

---

### 💡 Principais insights

* O tráfego é o principal fator de aumento no tempo de entrega
* Regiões semi-urbanas apresentam maior ineficiência operacional
* Eventos (festivais) aumentam significativamente o tempo médio
* Existe alta variabilidade na performance das entregas

---

### ⚠️ Limitações

Durante a etapa de limpeza, foi aplicada uma filtragem mais agressiva para viabilizar a análise inicial, o que pode introduzir viés nos resultados.

Evoluções futuras incluem:

* Tratamento mais robusto de dados ausentes
* Análise estatística mais aprofundada
* Inclusão de testes de hipótese

---

### 🛠️ Tecnologias utilizadas

* Python
* Pandas
* Plotly
* Streamlit
* Git & GitHub

---

## English

### 📌 About the project

This project is an end-to-end data analysis of a food delivery operation, focused on identifying the main factors that impact delivery time and operational efficiency.

The solution includes:

* Data cleaning and preprocessing
* Exploratory data analysis
* Business metrics development
* Interactive dashboard creation
* Web deployment

🔗 Access the dashboard:
https://amendes-curry-company.streamlit.app/

---

### 🎯 Objective

Answer the following business question:

> Which factors most impact delivery time and where are the main operational bottlenecks?

---

### 📊 Key analyses

* Traffic density impact on delivery time
* City-level performance comparison
* Weather condition effects
* Delivery driver performance analysis
* Event (festival) impact on operations

---

### 💡 Key insights

* Traffic is the main driver of delivery delays
* Semi-urban regions show higher inefficiency
* Festivals significantly increase delivery time
* High variability exists in operational performance

---

### ⚠️ Limitations

A more aggressive data cleaning approach was applied to enable initial analysis, which may introduce bias.

Future improvements include:

* More robust missing data handling
* Statistical analysis and hypothesis testing
* Model-based insights
